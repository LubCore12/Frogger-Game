# 🚦 Frogger Game (Godot)

A simple 2D game built with the **Godot Engine** where your goal is to cross a busy road as fast as possible — without getting hit by cars.

---

## 🎮 Gameplay

You control a character trying to cross a road filled with moving vehicles.

### Objective:
- Reach the other side of the road 🏁
- Do it in the **shortest time possible ⏱️**
- Avoid getting hit by cars 🚗💥
- 
---

## 🕹️ Controls

| Key        | Action        |
|------------|--------------|
| W / ↑      | Move up      |
| S / ↓      | Move down    |
| A / ←      | Move left    |
| D / →      | Move right   |
| Space      | Try again    |

---

## ⚙️ How to Play

You have two options to run the game:

### 1. Download Executable (Recommended)
- [Download](https://github.com/LubCore12/Frogger-Game/releases) the ready-to-play executable file
- Run it directly on your system

### 2. Run from Source Code
1. Download or clone this repository
2. Install the **Godot Engine**
3. Open the project in Godot
4. Press **Run ▶️**
